# Pharaoh's Name Translator (고대 이집트어 번역기)

한국어/영문 이름을 로마자 표기 → 고대 이집트 상형문자 스타일로 변환해 카르투슈(Cartouche)에 표시하는 단일 페이지 웹앱입니다.

- 상단(Hero): 번역기(입력/로마자 편집/변환/다운로드·공유)
- 하단(스크롤): 역사/변환 로직/카르투슈 의미/FAQ/면책 등 정보성 섹션(AdSense/SEO용 콘텐츠 보강)
